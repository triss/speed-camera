Stage 4

Derive measurements

How it works: once we have a track, we turn it into useful facts: how many, how long, which way, or how fast.

Child version: if you know where a car was before and where it is now, you can ask, “how far did it go, and how long did that take?”

The webcam demo derives pixel speed only. Real-world speed still needs the locate/calibration stage.
