Classify lightly

How it works: lookout can often make a rough guess from simple clues like size, speed, and path before using any heavier model.

Child version: a big fast thing on the road is probably a vehicle. A small slow thing wandering in a garden might be an animal.

Demo

No live classifier demo yet. Classification is a design note, not implemented code: the current system records simple measurements that could later feed a classifier.

Where the code is

There is no classifier function yet. The current code represents future classification indirectly through preset metadata: measurements, mode, and locate fields in files under web/js/presets/. definePreset() in web/js/engine/preset.js registers those stubs and supplies stub measure() and summarise() functions.
