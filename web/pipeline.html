How it works

The pipeline

lookout watches one place with a fixed camera. It asks: what changed, where did it go, where is that on the ground, and what useful number can we make from it?

Explained for a child: imagine taking two photos of the same street. If a blue car appears in the second photo, that is a change. If the blue car keeps moving, we give it a name and follow it. If we know where the road is, we can say where the car is on the road. Then we can count it or work out its speed.

1. Detect differenceFind what changed in the picture. 2. Track identityKeep the same name on the same thing. 3. LocateTurn camera pixels into a place on the ground. 4. Derive measurementsMake useful numbers like count, dwell, and speed. 5. Classify lightlyUse simple clues before heavier models.
