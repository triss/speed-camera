Locate

How it works: a calibration tells lookout how a flat ground shape in the camera maps to real metres on the ground.

Child version: the camera photo is like a squashed drawing of the road. Calibration is the trick that unsquashes the drawing so we can use a ruler.

Demo

No live webcam demo here yet. A correct locate demo needs calibration points for this camera view, otherwise it would pretend pixels are real metres without proof.

Where the code is

Presets declare which locate backend they need with the locate field in web/js/engine/preset.js. Browser calibration and pixel-to-ground projection are still TODO.
