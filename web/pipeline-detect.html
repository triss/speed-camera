Stage 1

Detect difference

How it works: compare one camera frame with another. Pixels that changed a lot probably belong to something moving, arriving, or leaving.

Child version: it is like a spot-the-difference puzzle. The street is mostly the same, but the car moved, so that part lights up.

The web version turns each frame into greyscale first, because brightness changes are cheaper to compare than full colour. It subtracts the previous frame from the current frame. Small changes are ignored as noise; larger changes become a motion mask. The first demo shows that mask as a smooth box. The second demo shows the mask itself.

The webcam demo asks for camera permission and works on HTTPS or localhost.

Demo: Camera Downsampling

Camera frames can be large. This demo first draws the camera into a smaller processing image, then stretches that small image back up so you can see what the computer sees. Lower settings make bigger blocks and do less work. Higher settings keep more detail but do more work.

Start the webcam, then move the slider to change downsampling.
